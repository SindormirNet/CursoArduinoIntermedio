# Curso Arduino Intermedio
Ejercicios del curso Intermedio de Arduino

![Curso ESP8266](https://sindormir.net/sites/sindormir.net/files/cursos/curso-arduino-intermedio-texto-dch.png.jpg)

Mas info: https://sindormir.net/curso-arduino-intermedio

## Temario:

* Programación:
  * Instalación y uso de librerías
  * Memorias internas de Arduino: Usando la flash y la EEPROM
  * Emulación de teclado y ratón con Arduino Leonardo
  
* Hardware:
  * Arduino Leonardo: nuevas características y posibilidades
  * Arduino como programador de microcontroladores ATmega (Arduino ISP)
  * Cómo hacer un Arduino sin Arduino: Interactúa con el mundo por 6€
  * Control de motores: Servomotores, Paso A Paso y C.C.
  * Manejo de displays LCDs
  * Fabricación y uso de sensores capacitivos
